# 工作空间视图

进入一个工作空间后，六个标签对应六种看同一支团队的角度。本页详解其中四个常用视图；[竞赛](/guide/fusion)与[研究委员会](/guide/research-committee)见各自的指南。

## 聊天

默认视图，也是你与团队打交道的主界面。

- **状态条**——顶部显示工作空间/方向名、当前有几个 AI 在线、源码上下文数量、当前分支，以及一个诚实的**引擎可用性**提示：只有经检测确实能跑的引擎才标「可用」。若没有可用引擎，会给出「安装 AI 引擎」入口；若队长正在启动，会显示「正在准备队长…」。
- **计划清单**——编排器维护的计划固定在对话上方：**计划 · 队长维护 · 已完成/总数**。每一步带状态图标（完成/进行中/受阻/待办）与负责人（队长或某个 worker 角色）。全部完成时折叠为「计划完成」。
- **对话与输入**——自然语言输入框；一个「优化输入」按钮可让 AI 把你的话改写得更清晰（保留技术细节、不臆造需求）；可选择把消息发给某个成员或**全员**；可**打断**正在跑的成员；还可为本方向选**模型**（opus/sonnet/haiku 层级 + 思考强度 低/中/高/极高/最大——切换会重启队长，有二次确认）。
- **流内的派发与交付卡**——队长派出 worker 时，对话流里出现「派给 {角色}」卡片（点开进入该 worker 的抽屉）；worker 完成时出现「{角色} 交付完成」卡片。
- **成员栏**——右侧列出所有成员。编排器固定在顶部、带金色环与「队长/调度」标记；worker 是临时的。每行显示状态点、一句「此刻在做什么」（如「正在调用 Edit · 文件 · 0:42」）、未读数与「唤醒」按钮；出错的成员排到最前。底部「近况」汇总各 worker 的最新进展。

## 协作图

把成员之间的关系画成一张实时依赖图。

- **图例**：**已满足**（绿色实线，依赖的交接已就绪）、**等待中**（琥珀色虚线并动画，依赖尚未满足）、**派生**（灰色细箭头，父子派生关系）。
- **画布**：每个 agent 一个节点（含已退出的 worker，灰显为「已完成」），显示角色、`引擎 · id`、状态点、暂停标记，以及交接信号（`→ 键名` 绿色，或 `✗ 键名.error` 红色表示交接失败）。节点超过一定数量会出现缩略图与缩放控件。节点可点选、不可拖动。
- **详情面板**：选中节点后显示角色、引擎、**状态**（就绪 / 已终止 / 已暂停 / 启动中）、交接信号、依赖的键，并提供「打开 Agent 抽屉 / 唤醒 / 暂停·恢复」。
- **批量控制**：右上角的「全停 / 全恢复」作用于当前方向的全部成员（有二次确认），并显示已暂停的数量。

## 工作台账

编排器思考过程的只读呈现（队长是唯一写入者，你是读者）。

- **任务台账**——目标、假设、计划（步骤依赖图）。
- **进展状态**——当前步骤、派出去的活、卡点。
- **近况**——各 worker 的最新进展心跳，最新在前。
- **操作**——「刷新」与「压缩」（用小模型把台账就地摘要，报告省下的 token）。每张卡显示「最后更新 XX 前」。

## 录像

每个 agent 的终端会话都会被录成可回放的文件，本视图是该工作空间的录像库。

- **卡片**——显示 agent 标识、**实时/已完成** 徽标、一张首帧缩略图（列×行 + 时长）、开始时间，以及「播放 / 下载」。可按 agent/id 搜索、按角色过滤。
- **播放器**——全屏、无干扰。快捷键：空格 播放/暂停，←/→ 前后跳 5 秒，可单帧步进，Esc 返回。正在进行的录像只能播到已写入的部分。

要深入单个成员（终端、活动、消息、产出），点击它打开 [Agent 抽屉](/dashboard/agent-drawer)。
