# 界面总览

swarmx 的仪表盘围绕**工作空间**展开：左侧是全局导航，中间是工作空间与它的多个视图。本页帮你快速认识各个区域，后续页面再逐块展开。

## 顶栏

- **品牌**——左上角的 swarmx 标识，点击回到对话主页。
- **命令面板**——搜索按钮（快捷键 `⌘K` / `Ctrl K`），快速跳转与执行操作。
- **通知**——右上角的铃铛，有未读时显示红点，点开预览近期事件，可进入完整通知页。

当本地服务端意外中断时，顶部会出现一条**后端离线**横幅，提供重试/重启入口。

## 左侧导航栏

一列纵向导航，可展开（图标 + 文字）或收起（仅图标，悬停显示名称），状态会记住。各项都是独立页面：

| 导航 | 用途 |
|---|---|
| **对话** | 产品的主surface：工作空间里与编排器及成员的对话。 |
| **文件** | 浏览本地文件。 |
| **终端** | 开一个 shell。 |
| **MCP** | 为 agent 装工具。 |
| **定时** | 定时任务。 |
| **目标** | 目标模式。 |
| **任务** | 任务看板。 |
| **用量** | token 与成本。 |

底部是**设置**。有新版本时设置图标会带一个红点。

「对话」是核心，其余是围绕它的工具页（详见[工具页](/dashboard/tool-pages)与[设置](/dashboard/settings)）。

## 工作空间界面

进入一个工作空间后，界面分为三块常驻区域：

- **工作空间侧栏**——列出所有工作空间（可搜索），每个可展开显示它的[工作方向](/guide/directions-and-isolation)与源码上下文；顶部有「新建工作空间」按钮（见[创建工作空间](/dashboard/create-workspace)）。
- **频道头 + 标签栏**——显示当前工作空间与方向名；一个 **在线/离线** 指示灯反映实时连接；非主线且已隔离的方向会出现「合并到主线」按钮。
- **视图区**——六个标签，切换时其余区域保持不动。

## 六个视图

标签顺序固定，可用 `⌘1`–`⌘6` 切换：

| 快捷键 | 标签 | 内容 |
|---|---|---|
| ⌘1 | **聊天** | 与编排器对话、计划清单、成员栏（见[工作空间视图](/dashboard/workspace-views)）。 |
| ⌘2 | **协作图** | 成员依赖与等待关系的实时图谱。 |
| ⌘3 | **工作台账** | 编排器维护的任务与进展台账。 |
| ⌘4 | **竞赛** | [融合竞赛](/guide/fusion)。 |
| ⌘5 | **研究委员会** | [研究委员会](/guide/research-committee)。 |
| ⌘6 | **录像** | 各 agent 的终端会话回放。 |

点击任意成员会打开 [Agent 抽屉](/dashboard/agent-drawer)，查看它的终端、活动、录像、消息与产出。
